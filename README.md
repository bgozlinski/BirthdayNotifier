# Birthday Notifier

## Overview
The Birthday Notifier is a Python script designed to help you keep track of birthdays happening in the current week. It organizes the birthdays by the day of the week and accommodates for weekend birthdays by moving the notification to Monday.

## Features
- Notifies of birthdays happening in the current week.
- Moves weekend birthday notifications to Monday to ensure you never miss a birthday.
- Displays the birthday notifications in a user-friendly format, organized by the day of the week.

## Installation
1. Clone this repository to your local machine.
2. Navigate to the project directory.

## Usage
To use the Birthday Notifier, follow these steps:

1. Update the `test_users` list in the script with the actual data.
2. Run the script by executing `python script_name.py` in the terminal (replace `script_name.py` with the actual name of your script file).
3. The script will display the birthdays happening in the current week in the console.
